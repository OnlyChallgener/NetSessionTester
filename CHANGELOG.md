## 说明
- Android 的候选 AP 扫描受系统权限、定位/WiFi权限和扫描频率限制影响，粘连AP仍属于“疑似诊断”。
- 本包未包含 Gradle Wrapper，当前环境无法直接编译 APK；请在 Android Studio 或 GitHub Actions 中构建验证。
