# Changelog

## V1.0.6-internal build106

自测版，重点优化设置页、Ping 日志、NAT 信息展示和卡片拖动体验。

- 设置页合并“目标设置”和“测试模式”为“目标与模式”。
- 测试模式与隐私模式同排显示，隐私模式去掉冗长说明。
- 输入框增加统一小图标，协议选择改为统一下拉框样式。
- 网络信息卡支持折叠/展开，折叠后只保留核心摘要。
- Ping 日志持久化保存，至少保留最近 5 次测试记录。
- Ping 日志弹窗改为最近一次汇总 + 最近 5 次折叠记录。
- NAT 信息增加公网映射展示，NAT1 采用多节点端口保持兼容口径。
- 首页测试卡片与设置页卡片加入长按拖动排序自测框架。
- GitHub Actions 产物名更新为 NetSessionTester-V1.0.6-internal-build106-signed.apk。
