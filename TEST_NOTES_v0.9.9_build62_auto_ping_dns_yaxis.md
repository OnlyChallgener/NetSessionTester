# NetSessionTester v0.9.9 build62 测试版改动

本测试版不用于 GitHub Release 推送，主要用于本地验证以下改动：

1. 会话增长曲线 Y 轴档位优化
- 取消 0-200 档位。
- 新增 0-300 档位。
- 新增 10000-20000 自动选 20000。
- 新增 20000-30000 自动选 30000。
- 35000 以上按 5000 的倍数向上取整，最高不超过 65535。

2. Ping 测试改为 AUTO 模式
- AUTO 使用 200ms / 500ms / 1000ms 三档。
- 测试开始前 5 秒使用 200ms，方便观察初始延迟波动。
- 普通状态使用 500ms。
- 活动连接数达到 5000 以上时使用 1000ms，减少高负载干扰。
- 释放中停止 Ping 采样。

3. DNS 诊断增加国外备用方案
- 系统 DNS 仍然优先。
- 系统 DNS 无 AAAA 时先查国内备用：223.5.5.5 / 119.29.29.29。
- 国内备用仍无 AAAA 时再查国外备用：1.1.1.1 / 8.8.8.8。
- UI 中区分“国内备用AAAA可用”和“国外备用AAAA可用”。
- 压测阶段仍使用已解析 IP，不进行高频 DNS 查询。

4. 保留上一版安全机制
- FD Guard、统一释放、generation 防 socket 回流、网络切换自动完整检测等逻辑保持不变。
- versionName 保持 0.9.9，versionCode 保持 62。
