# NetSessionTester v0.9.9 build68 本地自测说明

本版本为本地自测包，不修改推送 update.json。

## 主要改动

1. 下载更新弹窗增加重新下载机制
   - 下载失败后显示“重新下载 / 打开 GitHub / 关闭”。
   - 重新下载会取消旧任务、清理未完成的 `.part` 文件，并从 0% 重新开始。
   - 下载中增加“取消下载 / 后台下载”。

2. 新增值策略调整
   - 默认新增值改为 128。
   - 128 走智能调速策略。
   - 用户修改新增值后，静默切换为手动上限策略，不弹额外提醒。
   - FD 保护、失败降速、无增长停止仍然强制生效。

3. 会话曲线增加 CPS 右轴
   - 蓝线：会话数，使用左侧 Y 轴。
   - 橙线：CPS，使用右侧 Y 轴。
   - 两条曲线共用时间 X 轴。
   - 日志与曲线仍保持 1 秒采样。

## 继承内容

- build67 公共 STUN 增强版 NAT 检测。
- build66 动态 FD 保护。
- 释放进度图形展示。
- 底部导航适配和更新忽略功能。
