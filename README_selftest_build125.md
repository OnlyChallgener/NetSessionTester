# NetSessionTester V1.1.15 build125 自测说明

本包暂不建议发版，主要用于验证 Ping 图表和统计区修复。

## 主要改动

- Ping 统计区改为单行横向滑动，新增“抖动”和“时长”小卡。
- 抖动算法：最近 50 个成功 RTT，计算相邻 RTT 绝对差平均值；丢包/超时不参与。
- Ping 图表信息栏去掉“延迟波形/自动高频”文字，改为横向滑动展示目标、实际频率、窗口和状态。
- 修复停止后仍显示“实时中”的问题。
- 开始下一次 Ping 测试时重置图表、抖动、时长和旧会话状态，避免接续上次曲线。
- 图表 X/Y 轴安全边距加大，减少圆角遮挡。
- Ping 历史标题栏支持横向滑动，并显示测试时长。
