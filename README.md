# NetSessionTester

**NetSessionTester（宽带会话测试器）** 是一款面向 Android 设备的 TCP 会话数测试与网络诊断工具，主要用于测试 IPv4 / IPv6 网络环境下的连接保持能力、NAT 行为、Ping 延迟、会话增长曲线和释放稳定性。

当前版本：**V1.0.3 build103 hotfix**

---

## 主要功能

### 1. TCP 会话测试

支持对指定目标地址和端口进行 TCP 会话保持测试，用于观察当前网络、路由器、运营商出口或目标服务端在大量连接下的表现。

支持：

* 仅 IPv4 测试
* 仅 IPv6 测试
* IPv4 / IPv6 分别测试
* 自定义目标地址
* 自定义端口
* 自定义目标 CPS
* 自定义调度间隔 ms
* 固定 CPS 定速发射
* 会话增长曲线
* 峰值 / 最高值统计
* 失败区间统计
* 测试历史记录
* 测试结果导出

---

### 2. 固定 CPS 定速发射

新版核心采用 **定速发射模式**，不再使用动态 CPS / 智能调速。

运行中状态显示：

```text
正在运行 · 定速发射 · UI/曲线每秒采样
```

发射逻辑：

```text
每批发起数量 = 目标CPS × 调度间隔ms / 1000
```

示例：

```text
目标 200/s，调度间隔 100ms → 每批 20 条
目标 500/s，调度间隔 100ms → 每批 50 条
目标 1000/s，调度间隔 100ms → 每批 100 条
```

该模式可以更清晰地观察不同 CPS 设置下的会话增长能力。

---

### 3. IPv4 / IPv6 网络信息检测

App 会自动检测当前网络环境，包括：

* 本地 IP
* 公网 IPv4
* 公网 IPv6
* IPv4 / IPv6 优先级
* NAT 类型
* NAT 置信度
* 映射行为
* 过滤行为
* DNS AAAA 解析状态
* 当前网络类型
* 运营商识别

网络信息页支持手动刷新。

---

### 4. NAT / STUN 诊断

内置公共 STUN 节点检测逻辑，用于辅助判断 NAT 类型和 UDP 行为。

可显示：

* NAT 类型
* 端口保持 / 端口变化
* 端口受限 / 开放 / 待确认
* NAT 置信度
* STUN 节点成功数量
* 多轮复测稳定性

说明：

```text
公共 STUN 多节点推断仅作为辅助判断。
如需完全确认，需要 RFC5780 或自建 STUN 服务器复测。
```

---

### 5. Ping 测试

内置 Ping 延迟监控，用于观察压测前后网络稳定性。

支持显示：

* 当前延迟
* 平均延迟
* 最高延迟
* 最低延迟
* 丢包率
* 延迟曲线
* 自动刷新状态

Ping 图表会持续刷新，用于辅助判断压测是否影响网络质量。

---

### 6. 会话曲线

测试过程中会绘制会话增长曲线。

当前策略：

* 只保留会话数蓝线
* 不绘制 CPS 曲线
* 不绘制失败曲线
* 失败仅显示为区间文字
* UI / 曲线每秒采样
* 释放期间冻结曲线，避免下降线影响观感

失败区间示例：

```text
失败区间：10-20s +128，20-30s +232
```

---

### 7. 释放保护

测试结束或触发保护后，App 会进入释放流程。

释放界面显示：

* 释放进度
* 已释放数量 / 总数量
* 释放速度
* 预计剩余
* 释放完成耗时
* 释放步骤状态

释放完成后显示：

```text
耗时 1.6s
```

释放期间会冻结曲线渲染，只保留进度刷新，避免 UI 卡顿和按钮状态不同步。

---

### 8. FD 保护

为了避免 Android 系统文件描述符接近上限导致闪退，App 内置 FD 安全保护。

默认策略：

```text
活动会话接近 32360 时自动停止新增并释放
```

该保护用于防止连接数继续增长导致系统异常或 App 崩溃。

---

### 9. 失败终止机制

App 会根据当前会话规模判断失败数是否过高。

当前分段逻辑：

```text
0 - 1000        失败上限 120
1000 - 6000     失败上限 200
6000 - 12000    失败上限 360
12000 以上      失败上限 600
```

当失败达到当前区间阈值时，会停止新增连接并进入释放流程。

---

### 10. 更新检测

App 支持在线检测 GitHub Release 更新。

支持：

* 检测最新版本
* 一键更新
* 打开 GitHub
* 顶部下载浮层
* 下载完成后安装
* 忽略指定版本
* update.json 远程配置

当前发布文件命名规则：

```text
NetSessionTester-V1.0.3-build103-signed.apk
```

当前 Release Tag 示例：

```text
v1.0.3-103
```

## 使用建议

### 推荐基础设置

```text
目标CPS：200/s
调度间隔：100ms
```

适合稳定起步测试。

### 推荐中等测试

```text
目标CPS：500/s
调度间隔：100ms
```

适合观察普通路由器或宽带环境的会话增长能力。

### 推荐高强度测试

```text
目标CPS：1000/s
调度间隔：100ms
```

适合测试手机、路由器、宽带出口或自建服务器的高并发连接能力。

---

## 注意事项

高 CPS 测试会产生大量 TCP 连接，请仅用于：

* 自己的服务器
* 内网测试节点
* 明确授权的测试目标
* 自己的路由器 / 宽带环境

不建议长时间对公共网站进行高 CPS 压测。公共网站可能存在连接数限制，测试结果可能反映目标站限制，而不一定是本机或宽带真实上限。

## 项目定位

NetSessionTester 主要用于个人网络测试、宽带会话能力观察、IPv4 / IPv6 连接保持测试、NAT 行为判断和 Android 端高并发 TCP 会话实验。

该工具偏向网络诊断和个人测试，不是通用压力测试工具，也不建议用于未授权目标。
