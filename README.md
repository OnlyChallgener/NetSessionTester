# NetSessionTester

当前版本：V1.1.1 build111

## V1.1.1 重点更新

- 保留 V1.0.9 高频 Ping 引擎。
- Ping 图表改为主区间动态 Y 轴，避免偶发高延迟把曲线压平。
- 高延迟用橙色异常点标记，timeout 用红点标记。
- 图表显示目标地址、实际协议、最近 120 秒和实际频率。
- Ping 响应日志每次最多保存 1200 条，保留最近 5 次测试。
- Ping 日志弹窗显示存储用量和保存条数。
