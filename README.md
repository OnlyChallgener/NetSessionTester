# NetSessionTester V1.1.15 build131 自测包

本包用于自测，不建议直接发版。

## 重点改动
- Ping 间隔自动匹配推荐超时：30→300ms、50→500ms、100→800ms、200→1000ms、500→1500ms、1000→3000ms。
- Ping 统计区增加“超时：XX次”。
- 高频 TCP Socket Ping 采用固定节拍 + 并发上限 + 跳过保护。
- 并发满时主动跳过，不计入丢包。
- 优化断点绘制：丢包/null 不参与连线，过滤孤立短线段，减少毛刺。
- 网络信息与工具入口小图标语义化优化，IPv4/IPv6 增加 4/6 角标。
