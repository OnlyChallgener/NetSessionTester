# NetSessionTester

当前版本：V1.0.5-internal build105

本自测版重点优化 Ping 模块显示和日志体验，不改动 TCP 会话定速发射核心。

## 本版重点

- Ping 协议选择改为单行下拉，避免按钮换行。
- Ping 采集和界面展示分离：高频采集仍按用户设置执行，UI 与图表按秒聚合刷新。
- Ping 图表改为每秒聚合点，减少数字闪烁。
- Ping 图表蓝线改为平滑曲线。
- Timeout 不参与蓝线连接，仅显示底部红点。
- 采样中断或间隔过长时不强行连线。
- X 轴最多 6 个刻度，避免拥挤。
- Ping 日志保留弹出卡片式。
- Ping 日志按每次开始 Ping 分组，组头显示目标、协议、间隔、超时。
- 单条日志不再重复显示目标域名/IP。

## 自测建议

建议重点测试：

- Ping 间隔 30ms / 100ms / 1000ms 下 UI 是否稳定。
- Ping 日志分组是否正确。
- X 轴刻度是否不会超过 6 个。
- Timeout 后蓝线是否不会出现大斜线。
