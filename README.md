# NetSessionTester

当前版本：V1.0.4-internal build104 自测版

本自测版主要针对 NAT 识别和 Ping 模块进行验证，暂不建议作为正式 Release 发布。

## 本版重点

### NAT 检测

- IPv4 / IPv6 分开检测与展示。
- 公共 STUN 多节点复测继续保留。
- NAT1 判定采用兼容口径：公网 IP 一致、端口保持、多节点稳定时直接显示 `NAT1 / 全锥形`。
- 对称型仍按 `NAT4 / 对称型` 显示。
- 过滤行为不再显示“多节点推断”这种方法名，改为显示结果，例如 `开放型`、`未完成检测`、`端口受限`。
- 说明区保留 RFC5780 / 自建节点验证提醒，避免误导。

### Ping 独立运行

- Ping 从会话测试中独立出来，可单独开始 / 停止。
- 支持设置 Ping 目标、间隔、次数、超时、协议模式。
- 支持 Ping 响应日志。
- Ping 图表和会话图表互不强绑定。

### UI

- Ping 卡片增加开始、停止、日志、清空操作。
- 设置页新增 Ping 参数卡片，继续保持 Material3 + OneUI 风格。
- 保留 V1.0.3 的定速发射核心、FD 32360 保护、释放进度和释放耗时显示。

## 推荐自测

- NAT：WiFi / 蜂窝分别刷新网络信息，对比 My NAT、FastNet、checkmynat 结果。
- Ping：分别测试 1000ms、500ms、200ms 间隔下的稳定性。
- 会话：确认 200 / 500 / 1000 CPS 定速发射仍然准确。

## 说明

高 CPS 会产生大量 TCP 连接，仅建议用于自己的网络、服务器或明确授权目标。
