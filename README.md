# NetSessionTester V1.1.15 build130 自测包

本包用于自测，暂不发版。

## 本次重点
- Ping / 连接数测试目标预检：支持粘贴 URL，自动提取 host；错误地址会提示并不开始测试。
- 预检阶段不计入 Ping 时长、平均速率和曲线。
- 普通 Ping 测试进入后台/锁屏时按“准确优先”中断并保存部分记录，避免后台暂停后图表被画成直线。
- 后台/锁屏中断会在 Ping 历史中标记“中断”。
- 连接数测试仍沿用前台服务逻辑；Ping 同步监测继续跟随连接数测试目标。
