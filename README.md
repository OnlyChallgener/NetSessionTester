# 宽带会话测试器 NetSessionTester v0.3

安卓原生宽带 TCP 会话保持测试工具，用来测试自有/授权目标下 IPv4、IPv6 能稳定保持多少条 TCP 会话。

## v0.3 更新

- 去掉首页大段“使用提醒”，界面更简洁。
- 默认“每批新增”改为 100。
- 新增隐私保护开关：IP / 公网地址显示和导出时可打码。
- IPv4 / IPv6 分别测试和分别统计。
- 失败原因细分：超时、拒绝、重置、网络不可达、本地端口耗尽、Android FD/Socket 上限等。
- 修复状态栏图层/颜色，避免手机顶部状态栏看不清。
- 增加前台服务通知，测试运行或连接保持时降低被系统回收概率。
- 导出日志/CSV，包含结果汇总和日志。
- 统计项保持“新增”和“CPS”显示，和你参考的 Windows 工具口径一致。

## 默认参数

- 端口：80
- 测试模式：IPv4 / IPv6 分别测试
- 每批新增：100
- 间隔：500 ms
- 超时：3000 ms
- 失败停止：200
- 目标成功会话数：65535
- 测试完成后保持连接：开启

## 使用建议

请只测试自有 VPS、内网设备、路由器或已授权服务器。测宽带会话数建议慢慢累加，不要把每批新增和间隔设置得过猛，否则结果容易变成手机/目标服务器的瞬时建连压力。

推荐起步：

- 稳定测试：每批新增 100，间隔 500ms
- 快速测试：每批新增 200，间隔 500ms
- 激进测试：每批新增 500，间隔 500ms
