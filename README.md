# NetSessionTester V1.1.15 build129 自测版

本包继续优化 Ping 高频测试体验：

- 信息栏格式调整为：`192.168.0.1 · 抖动2.0ms · 瞬时30.3/s · 平均22.8/s · 窗口01:00 · TCP:80 · 实时`。
- X 轴在测试未满 1 分钟时动态显示 15s / 30s / 60s 窗口，避免右侧大片空白。
- 超过 1 分钟后默认显示最新 1 分钟窗口，拖动可查看历史，双击回实时。
- 高频 TCP Socket 探测改为：准备阶段不计入时长、端口只探测一次、固定节拍调度、允许少量并发。
- 瞬时速率和平均速率分开显示，避免启动预热拖低实时判断。
