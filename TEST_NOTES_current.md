# V1.0.6-internal build106 自测重点

1. 设置页“目标与模式”卡片是否正常保存地址、端口、测试模式、隐私模式。
2. Ping 参数协议框是否统一为下拉样式，不再换行或样式突兀。
3. Ping 开始/停止后日志是否持久化保存，重启 App 后至少保留最近 5 次记录。
4. Ping 日志弹窗是否显示最近一次汇总，最近 5 次记录是否可折叠/展开。
5. 网络信息卡右上角折叠/展开是否正常，折叠状态下摘要是否清晰。
6. NAT1 / 全锥形显示是否更接近 My NAT / FastNet 兼容口径，同时保留“过滤行为未验证”。
7. 首页和设置页卡片长按拖动排序是否无明显重叠、闪烁和卡顿。
8. TCP 会话测试核心、定速发射、FD保护、释放流程不应受影响。
