# build136 自测重点

1. Ping 只跑 5s、20s、60s，图形是否铺满且无大面积右侧留白。
2. RTT 集中在 40-70ms 时，Y 轴是否自动缩小，波形是否更明显。
3. 历史页展开图表后左右滑动，不应闪退。
4. Ping 响应日志是否出现“运行秒”列。
5. update.json status=draft 时，APP 不应弹正式更新卡片。
6. Release APK 未上传/404 时，不应弹可下载更新卡片，手动检测应提示安装包暂未发布。
