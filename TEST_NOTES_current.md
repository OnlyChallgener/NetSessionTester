# v0.9.9 build77 本地自测版

- 手动 CPS 模式提高优先级：用户设置 800/1000/1500 后，不再因普通失败率、增长效率、顶部确认把 CPS 压低。
- 手动模式只允许 pending 暂停、FD/资源硬停、低容量直接停止，不再低 CPS 慢跑。
- 低容量快速收尾：峰值低于 800、增长停滞且失败明显时，约 6 秒后直接停止新增。
- FD 接近快速收尾：手动模式 active >= 32200 或 fdRemaining <= 500 直接停止新增并释放。
- UI 快照节流到约 500ms，减少测试收尾和 IPv4/IPv6 切换时的重组卡顿。
- 继承 build76：默认 128 智能调速、IPv4 6000+ 高容量策略、IPv6 高容量策略、时间发射器、释放进度、更新重试、STUN 增强。
