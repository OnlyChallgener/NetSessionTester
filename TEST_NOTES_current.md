# NetSessionTester v0.9.9 build84 自测版

本版基于 build83 / v0.9.7 高速测速核心，只修收尾和统计问题，不再重写测速核心。

## 重点改动

1. FD / Socket 上限提前保护
   - active 或 peak >= 32000 直接按 Android FD / Socket 上限保护停止新增并释放。
   - active >= 31800 后会裁剪本轮新增数量，避免一次大批量超冲到 32500+ 导致闪退。
   - active >= 32200 作为紧急兜底。

2. 修复接近失败上限时 CPS 掉底僵直
   - 取消 failureBudget 把 chunk 缩到 1 个连接的逻辑。
   - 接近失败上限时仍保持整批并发发起；失败展示由外层按阈值裁剪，命中后直接收尾。

3. CPS 改成真实墙钟统计
   - 不再用固定 intervalMs 估算 CPS。
   - CPS = 本轮新增数 / 实际耗时，避免显示 800/1000 但实际跑很久的错觉。

4. 低容量更快收尾
   - peak < 300、运行 5 秒、增长停滞且失败 >= 40 直接收尾。
   - peak < 1000、运行 6 秒、增长停滞且失败 >= 80 直接收尾。

## 保留

- v0.9.7 批量高速测速核心
- 分段失败阈值：120 / 200 / 360 / 600
- 运营商判断
- NAT 公共 STUN 增强版
- 释放图形进度
- 下载失败重试
