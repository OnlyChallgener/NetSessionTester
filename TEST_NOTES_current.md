
# Build119 MTU 自测重点

- 综合模式应同时显示本地 MTU、TCP 业务、应用层 PLPMTUD说明、ICMP路径结果。
- 本地模式不应发网络探测包，只显示本机链路 MTU。
- TCP模式应验证 443 连接，并说明当前为 Java 层估算 MSS，NDK TCP_MAXSEG后续增强。
- ICMP模式应继续显示二分探测过程。
- PLPMTUD模式应提示需要专用测试节点，不应伪造结果。

2. 漫游测试运行时，网络事件区域应显示 ConnectivityManager 监听到的网络可用/能力变化/链路属性变化。

4. Ping 表 / 信号表竖线应对应 BSSID 切换事件。

8. 测试标准 1s、高频 500ms、低频 2s 采样切换。
9. 继续观察小按钮/卡片点击水波纹和阴影是否还有方形扩散。
