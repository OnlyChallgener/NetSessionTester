# NetSessionTester V1.1.15 build126 自测说明

本包继续优化 Ping/连接数测试联动与历史体验：

- “测试控制”改为“连接数测试”。
- 连接数测试优先级高于 Ping；开始连接数测试时会停止旧 Ping 并重新开始同步 Ping 监测。
- 抖动数据加入 Ping 图表小字信息栏靠前位置，并保留统计区抖动小卡。
- Ping Y 轴增加 0-10ms / 0-20ms 分档，路由器低延迟波动更明显。
- X 轴不再显示负数，统一显示测试经过时间。
- 图表默认显示最新 1 分钟窗口，随测试时间滚动推移；拖动可查看历史，双击回实时。
- Ping 历史改为二级页面，最多保留 8 条记录，默认折叠，点击展开可查看可拖动图形和总结。
- 历史标题栏支持横向滑动，并显示时长。
